## ASCII コードについて

**ASCII**（アスキー）は文字と数値を対応させる規格で、コンピュータが文字を扱うときの基本になっている。

アルファベット・数字・記号など「印字できる文字」は ASCII コードと pygame のキー定数の値が一致する。

```python
print(pg.K_a)      # → 97  （ASCII の 'a' と同じ）
print(pg.K_0)      # → 48  （ASCII の '0' と同じ）
print(pg.K_SPACE)  # → 32  （ASCII の スペース と同じ）
```

これを利用すると、`ord()` 関数でキーの文字コードを求められる。

```python
# 'a' の ASCII コードを確認する
print(ord('a'))    # → 97

# event.key と比較するのに使える
if event.key == ord('a'):
    print("a キーが押された")
```

> **矢印キーやファンクションキーは ASCII 範囲外**  
> `pg.K_UP`（1073741906）のように大きな値になる。これらは SDL2 のキーコードで、ASCII とは無関係。

### 主な ASCII コード

| 値 | 文字 | 値 | 文字 | 値 | 文字 |
| --- | --- | --- | --- | --- | --- |
| 32 | スペース | 65〜90 | A〜Z（大文字） | 97〜122 | a〜z（小文字） |
| 48〜57 | 0〜9 | 33〜47 | `! " # $ % & ' ( ) * + , - . /` | 58〜64 | `: ; < = > ? @` |

> **修飾キーの組み合わせ判定**
> 修飾キーが押されているかは `pg.key.get_mods()` で確認できる
> ```python
> mods = pg.key.get_mods()
> if mods & pg.KMOD_SHIFT:   # Shift が押されている
>     print("Shift + 何かのキー")
> ```

| 定数 | 意味 |
| --- | --- |
| `pg.KMOD_SHIFT` | 左右どちらかの Shift |
| `pg.KMOD_CTRL` | 左右どちらかの Ctrl |
| `pg.KMOD_ALT` | 左右どちらかの Alt |
