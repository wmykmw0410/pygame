# ゲームを作る7ステップ
```text
1.ゲームの準備をする

2.この下をずっとループする

    3.画面を初期化する

    4.ユーザからの入力を調べる ※ 閉じるボタンの検出（ステップ7）もここで行う

    5.絵を描いたり、判定したりする

    6.画面を表示する

    7.閉じるボタンが押されたら、終了する（ステップ4のイベント処理に含まれる）
```

## ステップ1.ゲームを準備する
```python
# ライブラリのimportをする
import ライブラリ名

# importしたライブラリやモジュールに省略名をつける
import ライブラリ名 as 省略名


# pygameでは最初に初期化を行う
pg.init()

# 初期化をしたらゲーム用のウィンドウを作る(横800 x 縦600)
# 戻り値は変数screenに入れる
# このプログラムはscreenに対してゲーム画面を描いていく
screen = pg.display.set_mode((800, 600))

# ウィンドウのタイトルバーに表示する文字列を設定する
pg.display.set_caption("タイトル名")

# FPS(Frames Per Second)を管理するClockオブジェクトを作る
# FPSとは1秒間に画面を何回更新するかを表す値
# 値が大きいほど滑らかになるが、処理負荷も上がる
clock = pg.time.Clock()
```

```python
# ゲームをずっとループする部分
# 繰り返しの中で「閉じるボタンが押されたら終了する」処理を入れる
while True:

```

## ステップ3.画面を初期化する
```python
# 繰り返しの中に入ったら、初めに画面をきれいに消す
# screenにfill関数で塗りつぶしを行う
# pg.Color("WHITE")で白を指定して、真っ白に塗りつぶす
screen.fill(pg.Color("WHITE"))
```

## ステップ4.ユーザからの入力を調べる
```python
# キーボードやマウスから発生したイベントをpg.event.get()で受け取り
# for文で1つずつevent変数に取り出して調べる
# ※ 閉じるボタンの検出（ステップ7）もここで行う
for event in pg.event.get():
    if event.type == pg.QUIT:
        pg.quit()
        sys.exit()
```

## ステップ5.絵を描いたり、判定したりする
```python
# 真っ白になったゲーム画面に、赤い四角形を描く
# pygameのdraw.rect関数で四角形を描く
pg.draw.rect(screen, pg.Color("RED"), (100, 100, 100, 150))
pg.draw.rect(screen, pg.Color("色"), (x座標, y座標, 幅, 高さ))
```

## ステップ6.画面を表示する
```python
# 画面に描くという処理はコンピュータにとって時間がかかるため
# 裏側（メモリ内）で絵を描いておき（screen）、全部できてから一気に表示させる
pg.display.update()

# clock.tick(fps)でFPSを制御する
# 例えばclock.tick(60)とすると、1秒間に最大60回ループが回るよう速度を調整する
#   デフォルト値: 0（制限なし、できる限り高速で実行）
#   範囲: 0以上の整数（推奨: 30〜60）
clock.tick(60)
```

## ステップ7.閉じるボタンが押されたら、終了する
```python
# ステップ4のイベントループ（for event in pg.event.get()）の中に書く
# ユーザがウィンドウの「閉じる」ボタンを押すと
# 「何かが押された」というお知らせがくる（イベント）
# 「押されたものが何か（event.type）」を調べて
# pg.QUITであれば pg.quit()でpygameを終了し、sys.exit()でプログラムを終了する
if event.type == pg.QUIT:
    pg.quit()
    sys.exit()
```
