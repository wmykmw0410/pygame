# ステップ7.閉じるボタンが押されたら、終了する
```python
"""ループを抜けるための終了処理を入れる
ユーザがウィンドウの「閉じる」ボタンを押すと、
pythonには「何かが押された」というお知らせがくる(イベント)
「押されたものが何か(event.type)」を調べて、pg.QUITであれば
「閉じる」ボタンが押されたことがわかるので、終了処理を行う
pg.quit()でpygameを終了し、sys.exit()でウィンドウを終わらせる
for event in pg.event.get():はイベントを調べる繰り返し文である
キーボードやマウスから発生したイベントをpg.event.get()で受け取り、
for文で1つずつevent変数に取り出して調べている"""
for event in pg.event.get():
    if event.type == pg.QUIT:
        pg.quit()
        sys.exit()
```
